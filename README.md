<h3>🔧Installation & Run Project<h3/>

<h3>Requirements:</h3>
Node.js >= 16

MongoDB
Vite
<h3>Run frontend:</h3>
cd frontend
  
npm install

npm run dev
<h3>Run backend:</h3>
cd backend

npm install

npm start
